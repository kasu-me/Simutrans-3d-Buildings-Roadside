# 3Dロードサイド建築物セット
Magica Voxelで制作したSimutrans(pak128系)向けの市内建築アドオンです。

## はじめに
ロードサイド的な市内建築物のアドオンのうち、公開せずお蔵入りになっていたアドオンたちをまとめて公開します。公開用に作ったものではないため様々な粗がありますがお許しください。

## 内容物
- 快活クラブ
- ドンキホーテ
- タイヤ館
- 都市型ラブホ
- その他雑居ビル等各種市内建築物

それぞれ、pak･png･dat･voxファイルを収録しています。
